= ZEN OBJECT VIEWER

== Build & Install

[source, shell]
----
$ git clone https://github.com/zigen-project/zen.git
$ cd zen
$ meson build
$ ninja -C build install
----

== Run

=== STL File viewer

[source, shell]
----
$ zen-object-viewer [STL FILE]
----

[source, shell]
----
$ zen-object-viewer ./assets/teapot.stl
----

You can move the model by `left click + motion`
